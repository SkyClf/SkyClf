Quick Start

1 Clone the repository 

git clone https://github.com/SkyClf/skyclf.git
cd skyclf

2 Configure your AllSky camera URL 

# Edit docker-compose.yml
- SKYCLF_ALLSKY_URL=http://your-camera-ip/latest.jpg

3 Start with Docker 

docker compose up -d

4 Get predictions 

curl http://localhost:8080/api/clf

API Endpoints

GET /api/clf

Simple classification response with skystate and confidence. 

{
  "skystate": "clear",
  "confidence": 0.9847,
  "probs": {...}
}
GET /api/latest

Full details including image metadata, labels, and prediction. 

{
  "status": "ok",
  "image": {...},
  "label": {...},
  "prediction": {...}
}
GET /latest.jpg

Serves the latest fetched AllSky camera image directly.

GET /health

Health check endpoint for monitoring and load balancers.
